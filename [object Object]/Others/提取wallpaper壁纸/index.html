# 提取 wallpaper 静态壁纸

电脑配置很低，如果一直开着 WallpaperEngine 也是十分消耗资源，于是想着将好看的静态壁纸提取出来，上网搜了一下发现大家都在使用 repkg 来实现，于是就有了这篇文章。

# 一、准备工作

1. 下载 repkg，Github 链接。

2. 准备好需要提取的壁纸。

# 二、使用步骤

1. 把下载的 repkg.exe 放入一个空文件夹，可以在文件路径下新建一个输出文件夹，命名为 output。

2. 在 repkg 文件夹下按住 shift 点击右键，打开 powershell。

3. 打开需要提取的壁纸文件夹，找到对应的 pkg 文件。


4. 在 powershell 中输入 .\repkg.exe extract , 然后把 pkg 文件拖入 powershell 窗口中，补上命令 -o .\output 后回车（注意箭头处要有空格）。

5. 成功后会在 output 文件夹下生成一个壁纸文件夹，里面有提取出来的壁纸。
